Project Overview

KARIB is a specialized web application designed to uphold and elevate workplace standards regarding Neatness, Aesthetics, and Cleanliness (Kerapian, Keindahan, Kebersihan). By digitizing the evaluation process, the system aims to foster a culture of discipline, creating a comfortable and professional environment for both employees and visitors.

The platform facilitates weekly assessments conducted by a designated evaluation team. Beyond simple data entry, KARIB serves as an analytical tool that automatically aggregates scores to identify the highest and lowest-performing employees.

Key Features

Granular Performance Reporting

Automatically generates score recaps and rankings on a weekly, monthly, and quarterly basis to monitor progress effectively.

Role-Based Access Control (RBAC)

Distinct access levels for 'Evaluators' (to input scores) and 'Admins' (to manage settings and view full reports), ensuring data integrity.

Comprehensive User Management

A dedicated module for administrators to manage employee data and system access
